# Lista de Insumos y Goleadores

Este repositorio incluye un ejemplo de formulario de requerimiento de insumos y una sencilla interfaz web para administrar una **tabla de goleadores** por categoría.

## Tabla de goleadores

El archivo `scoreboard.html` contiene una pequeña aplicación en la que es posible:

- Registrar un jugador con su DNI, categoría (Fútbol 7, Fútbol 9, Fútbol 11 o Sub 20), equipo y cantidad de goles marcados.
- Si un jugador ya existe en la categoría, los goles se acumulan.
- Eliminar jugadores de la tabla.
- Los datos se guardan en `localStorage` para conservarse entre recargas del navegador.

Para usar la aplicación, abre `scoreboard.html` en un navegador web moderno. La interfaz está en español y no requiere ningún servidor adicional.
